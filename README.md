# Vibe Coding Club

AI와 함께하는 가장 트렌디한 코딩 커뮤니티

## 소개

Vibe Coding Club은 AI를 활용한 다양한 활동을 함께 경험하고, 학습하며, 도전하는 커뮤니티입니다. 난이도에 따라 Creative / Develop 두 개의 트랙으로 운영되며, 매주 토요일(Creative)과 일요일(Develop)에 활동이 진행됩니다.

## 기술 스택

### Frontend
- **Next.js 16** - React 기반 풀스택 프레임워크
- **React 19** - UI 라이브러리
- **TypeScript** - 타입 안전성
- **Tailwind CSS v4** - 유틸리티 퍼스트 스타일링
- **Framer Motion** - 애니메이션

### 개발 도구
- **Turbopack** - 빠른 번들링
- **ESLint** - 코드 품질 관리
- **Prettier** - 코드 포맷팅

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 프로덕션 실행
npm start
```

## 바이브코딩 (Vibe Coding)

이 프로젝트는 **바이브코딩** 방식으로 개발되었습니다.

바이브코딩이란 AI 코딩 에이전트와 함께 대화하며 코드를 작성하는 새로운 개발 방식입니다. 개발자가 원하는 기능을 자연어로 설명하면, AI가 코드를 생성하고, 개발자는 이를 검토하고 피드백을 주는 협업 과정을 통해 빠르고 효율적으로 프로젝트를 완성합니다.
